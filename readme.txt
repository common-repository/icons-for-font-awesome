== Description ==


An easy way to place Font Awesome icons on your Wordpress site.  Many of the other plugins used for placing Font Awesome Icons on your site are not kept up to date with the latest stable Font Awesome release.  We are providing this plugin to establish a plugin in the repository that has the most up to date stable release of Font Awesome.

###Quick Start Guide

* [font awesome icons quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/font-awesome-icons/)

###What you get when you use the Font Awesome Icons

*  Place Font Awesome Icons using the standard Font Awesome code
*  All of the necessary Font Awesome code is included on all of your post and pages so you can use the Font Awesome icons when ever you want.
*  Use the standard Font Awesome Icon tag <i> to display a Font Awesome Icon (ex. <i class="fa fa-camera-retro fa-lg"></i>).


###Add font awesome icons to any post or page)

Whether you are writing a post about your company or another company, you can include font awesome icons.

###Quick Start Guide

* [font awesome icons quick start guide at thinklandingpages.com](http://www.thinklandingpages.com/font-awesome-icons/)

###Font Awesome Icons 

Add as many font awesome icons to your site as you like. 



== Installation ==


1. Upload `font-awesome-icons-wp` to the `/wp-content/font-awesome-icons-wp/` directory
1. Activate the font awesome icons through the 'Font Awesome Icons WP' menu in WordPress
1. Click **Font Awesome Icons WP** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The font awesome icons does the programming and design.

= Is there a limit to the number of font awesome icons I can create? =
No, unlimited font awesome icons can be created.

= Can the font awesome icons be put on my homepage? =
Yes, you can put font awesome icons on your homepage.

= Can I put font awesome icons on any post or page? =
Yes, you can put Font Awesome Icons on your pages and posts.
